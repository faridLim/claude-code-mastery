This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 언어 및 커뮤니케이션 규칙

- **기본 응답 언어**: 한국어로 진행
- **코드 주석**: 한국어로 작성
- **커밋 메시지**: 한국어로 작성
- **문서화**: 한국어로 작성
- **변수명/함수명**: 영어 사용 (코드 표준 준수)
- **문서 및 README**: 한국어로 작성

## 프로젝트 상태

이 프로젝트는 초기 단계입니다. 프로젝트가 발전함에 따라 이 파일을 업데이트하여 다음 정보를 추가하십시오:

### 추가될 항목 (프로젝트 진행 중)

1. **빌드 및 실행 명령어**
   - 프로젝트 빌드 방법
   - 개발 서버 실행 방법
   - 프로덕션 배포 방법

2. **테스트 및 린팅**
   - 테스트 실행 명령어 (전체, 단일)
   - 린트 검사 명령어
   - 코드 포맷팅 도구 및 명령어

3. **코드 아키텍처**
   - 고수준 프로젝트 구조
   - 주요 모듈 간의 관계
   - 데이터 흐름도 (필요시)

4. **개발 환경 설정**
   - 필수 의존성 및 버전 요구사항
   - 개발 환경 세팅 방법
   - 데이터베이스 또는 외부 서비스 설정

5. **배포 및 운영**
   - CI/CD 파이프라인
   - 배포 프로세스
   - 모니터링 및 로깅

## 개발 가이드라인

### 코드 스타일

- 변수, 함수, 클래스명은 영어로 작성
- 모든 주석과 문서는 한국어로 작성
- 영어와 한국어를 혼합할 때는 일관성 있게 사용

### Git 커밋

- 커밋 메시지: 한국어로 작성
- 형식: `[태그] 설명` (예: `[feat] 사용자 인증 기능 추가`)
- 자세한 설명이 필요한 경우 본문에 한국어로 기술

### 문서화

- 모든 README, 설명 문서는 한국어로 작성
- API 문서에는 한국어로 상세한 설명 포함
- 예제 코드는 필요시 추가
